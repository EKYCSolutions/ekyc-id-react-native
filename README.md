# ekyc-id-react-native
React Native Wrapper for EkycID native SDKs.
## Installation

```sh
npm install ekyc-id-react-native
```

## Usage

```js
import { EkycIdReactNativeView } from "ekyc-id-react-native";

// ...

<EkycIdReactNativeView color="tomato" />
```
